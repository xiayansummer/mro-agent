MRO 外部比价助手 · 隐私政策

最后更新：2026-06-14 · 适用于「MRO 外部比价助手」浏览器扩展（Chrome / Edge）

「MRO 外部比价助手」是 MRO Agent（https://mro.fultek.ai）的配套浏览器扩展。它在你主动发起比价时，用你已登录的京东工业品 / 震坤行账号打开对应平台的搜索结果页、采集前若干条在售商品信息，回传给 MRO Agent 做归一、排序与比价。本扩展单一用途，仅服务于工业品比价。

一、我们处理哪些数据

数据用途去向
绑定令牌（extToken）、设备名、扩展版本把扩展与你的 MRO Agent 账号关联、上报在线状态发送至 mro.fultek.ai（后端仅保存令牌的 SHA-256 哈希）
平台登录态（已登录 / 未登录 / 未知）判断能否执行比价、引导你登录发送至 mro.fultek.ai
搜索结果页上的公开在售商品信息（标题、品牌、价格、单位、货期、商品链接、平台 SKU）生成比价结果发送至 mro.fultek.ai，展示在你的比价卡片中

二、我们不做什么

  • 不读取、不上传任何 Cookie
  • 不读取、不上传 localStorage / 浏览历史 / 表单数据
  • 不上传任何页面的 HTML 原文；只回传解析后的结构化比价条目
  • 不采集你的账号密码；登录态识别仅基于页面公开特征
  • 不向任何第三方出售或共享你的数据；数据仅用于你自己账号下的比价
  • 不加载或执行任何远程代码

三、权限说明

  • storage：在本机保存绑定令牌与设置
  • tabs + scripting：在后台标签页打开京东工业品 / 震坤行搜索结果页并解析其中的公开商品信息
  • alarms：定时（约每分钟）上报在线状态、领取待执行的比价任务
  • 主机权限 *.jd.com / *.jd.hk / *.zkh.com / mro.fultek.ai：仅限上述平台与本服务后端

四、数据安全与你的控制

  • 绑定令牌在后端仅以哈希形式存储，一个账号同一时间只保留一个在线扩展
  • 你可随时在扩展弹窗点「解除本机绑定」清除本机令牌，或在浏览器中卸载本扩展，即停止一切数据采集
  • 比价数据归属于你的 MRO Agent 账号，与你的账号隔离存储

五、联系方式

如对本隐私政策有任何疑问，请联系 MRO Agent 管理员，或通过 https://mro.fultek.ai 反馈。
